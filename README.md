## ✨ 是什么？

输入任意主题词，AI 就能生成一段**抖音/B站猫meme风格**的作文文案。

适用于：
- 📱 短视频配音文案
- 💬 小红书 / 评论区互动
- ✍️ 粉丝创作素材
- 🎓 学生党文案灵感

## 📖 风格解析

一篇合格的猫meme作文，长这样：

### 1️⃣ 意象开场
用具体事物切入，引发共鸣

```
镜子很脏的时候，你并不会以为是自己的脸脏。
那为什么别人说出糟糕的话语时，你就要觉得糟糕的是自己呢？
```

### 2️⃣ 哲理反转
通过**不是…而是…** / **后来才明白…** / **原来…**制造反转

```
我们总被催促着要快一点，要赶在所有人前面，
却忘了偶尔停下来看看自己是否还在正确的路上。
```

### 3️⃣ 金句收尾
押韵 + 正能量 = 扎心

```
心有繁花，何惧敷衍。
往后余生，以自己的步调前行。
```

## 🎯 核心特征

| 特征 | 说明 |
|------|------|
| **短句为主** | 句子简洁，每句不超过15字，节奏感强 |
| **具象比喻** | 镜子、表、月亮、蝉鸣、晚风、星辰… |
| **哲理反转** | `不是…而是…` / `后来才明白…` / `原来…才…` |
| **押韵金句** | `心有…何惧…` / `纵无…亦…` / `…才是…` |
| **150-200字** | 恰到好处的长度，适合短视频 |
