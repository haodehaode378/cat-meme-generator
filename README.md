# 猫Meme作文生成器

抖音/B站风格的猫meme作文文案生成Skill，配合Claude Code使用。

## 简介

输入任意主题词，即可生成150-200字的猫meme风格作文文案。适用于短视频文案、评论区互动、粉丝创作等场景。

## 效果示例

### 输入
```
/猫meme 暗恋
```

### 输出
```
月亮从不开口说话
却照见了我所有的心事

我以为暗恋是一场哑巴戏
只有自己在演
后来才明白
月亮早就把你的名字
写进了每一个有我的夜晚

原来有些话不必说出口
风会替我打听
有些人不必靠近
光会替我拥抱

不必急着告白
有些温柔本身就是一种答案
```

更多效果参考 [cat-meme/](cat-meme/) 目录下的60篇范例。

## 使用方法

### 调用方式
```
/猫meme [主题词]
```

### 示例
```
/猫meme 孤独
/猫meme 深夜emo
/猫meme 暗恋
/猫meme 自卑
/猫meme 毕业
```

## 风格特征

- **三段式结构**：意象开场 → 哲理反转 → 金句收尾
- **短句为主**：句子简洁，节奏感强
- **具象比喻**：镜子、表、月亮、蝉鸣、晚风等意象
- **哲理反转**：`不是…而是…` / `后来才明白…` / `原来…`
- **押韵金句**：`心有…何惧…` / `纵无…亦…` / `…才是…`
